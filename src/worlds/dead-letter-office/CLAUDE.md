# The Dead Letter Office — Claude instructions

Undeliverable mail drifts down through lamplight; opened letters can be read, and a few return addresses still work.

## Docs

- `changelog.md` — session history, newest first.
- `README.md` — world overview and structure.

## World-specific rules

- Built on the Blender world-art pipeline: rendered plate + baked character (the postmaster,
  a Meshy rig with NLA-exported animations) + NDC hotspots. Re-rendering means going back
  through the Blender build in `tmp/dead-letter-office/`, not editing the PNGs.
- Nothing else documented yet — add constraints here as they surface.
